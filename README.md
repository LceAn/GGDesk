# GGDesk
桌面快捷方式启动程序。

# 计划
 - 具备免安装程序主程序扫描功能。
 - 桌面快捷方式扫描
 - Win商店程序扫描
 - 系统默认安装路径扫描

# 更新记录
 - [251117]：市面上的都不好看，也不好用，计划自己开发一个。

 - [20251118] Beta 3.0 - UI 重构与体验升级
   **新增功能**
   
   - **全新 UI 布局**：弃用垂直堆叠布局，采用现代化的 **侧边栏 (Sidebar) + 多页面 (Stacked Widget)** 布局。
   - **扫描控制**：分离“选择目录”与“开始扫描”操作，新增 **“停止扫描”** 功能，支持随时中断任务。
   - **图标支持**：程序列表中现在可以显示 `.exe` 文件的原生图标，视觉体验大幅提升。
   - **主题切换**：内置 **暗黑模式 (Dark)** 和 **明亮模式 (Light)**，用户可在设置中一键切换。
   - **自定义输出**：支持用户自定义快捷方式的生成路径（不再强制桌面）。
   - **交互升级**：支持 **双击** 列表项直接修改程序详情，移除晦涩的 "Stage 1/2" 术语。
   
   **🛠️ 优化与修复 **
   
   - **日志降噪**：主界面移除大段文本日志，改为 **底部状态栏 (Status Bar)** 显示关键进度。完整日志移至“应用设置”页面。
   - **性能优化**：扫描过程在独立线程 (`QThread`) 中运行，彻底解决 UI 卡死 (Not Responding) 问题。
   - **稳定性修复**：修复了多线程下对象生命周期管理不当导致的 `0xC0000409` 崩溃问题。
   
   **技术细节**
   
   - **Backend**: `discover_programs` 增加 `check_stop_callback` 回调支持中断。
   - **Frontend**: 引入 `QFileIconProvider` 提取图标；引入 `QSS` (Qt Style Sheets) 进行全局样式定义。
   - **Config**: 配置文件新增 `theme` 和 `output_path` 字段。
